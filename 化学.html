人教版高中化学必修一化学方程式汇总

第三章 金属及其化合物

一、金属的化学性质

  1. 钠在空气中变暗：4Na + O2 = 2Na2O （常温）
  2. 钠在空气中燃烧（黄色火焰）：2Na + O2 ≜ Na2O2 （艳色反应）
  3. 钠与水反应（浮、熔、游、响、红）：2Na + 2H2O = 2NaOH + H2↑
  4. 钠与盐酸反应：2Na + 2HCl = 2NaCl + H2↑
  5. 钠与CuSO4溶液反应：2Na + 2H2O + CuSO4 = Na2SO4 + Cu(OH)2↓ + H2↑
  6. 钠在氯气中燃烧（产生白烟）：2Na + Cl2 ≜ 2NaCl （点燃）
  7. 钠与硫的反应（爆炸）：2Na + S ≜ Na2S （研磨）
  8. 铝箔在氧气中剧烈燃烧：4Al + 3O2 ≜ 2Al2O3 （点燃）
  9. 铝与盐酸反应：2Al + 6HCl = 2AlCl3 + 3H2↑
  10. 铝与氢氧化钠溶液反应：2Al + 2NaOH + 2H2O = 2NaAlO2 + 3H2↑
  11. 铁在氧气中燃烧：3Fe + 2O2 ≜ Fe3O4 （点燃）
  12. 高温下铁与水蒸气反应：3Fe + 4H2O(g) ≜ Fe3O4 + 4H2 （加热）
知识点：钠是非常活泼的金属，需保存在煤油中；铝是两性金属，既能与酸反应也能与碱反应；铁是变价金属，与不同氧化剂反应生成不同价态的化合物。

二、钠的重要化合物

  1. 氧化钠与水反应：Na2O + H2O = 2NaOH
  2. 氧化钠与二氧化碳反应：Na2O + CO2 = Na2CO3
  3. 氧化钠与盐酸反应：Na2O + 2HCl = 2NaCl + H2O
  4. 过氧化钠与水反应：2Na2O2 + 2H2O = 4NaOH + O2↑
  5. 过氧化钠与二氧化碳反应：2Na2O2 + 2CO2 = 2Na2CO3 + O2
  6. 过氧化钠与盐酸反应：2Na2O2 + 4HCl = 4NaCl + 2H2O + O2↑
  7. 碳酸钠与足量盐酸反应：Na2CO3 + 2HCl = 2NaCl + H2O + CO2↑
  8. 碳酸钠与少量盐酸反应：Na2CO3 + HCl = NaHCO3 + NaCl
  9. 碳酸氢钠与盐酸反应：NaHCO3 + HCl = NaCl + H2O + CO2↑
  10. 碳酸氢钠与氢氧化钠反应：NaHCO3 + NaOH = Na2CO3 + H2O
  11. 向饱和碳酸钠溶液中通入CO2气体会析出晶体：Na2CO3 + CO2 + H2O = 2NaHCO3
  12. 碳酸氢钠受热分解：2NaHCO3 ≜ Na2CO3 + H2O + CO2↑ （加热）
知识点：过氧化钠是淡黄色固体，可作为供氧剂；碳酸钠比碳酸氢钠稳定，碳酸氢钠受热易分解；碳酸钠与酸反应分步进行，取决于酸的量。

三、铝的重要化合物

  1. 氧化铝是两性氧化物：
    • 与盐酸反应：Al2O3 + 6HCl = 2AlCl3 + 3H2O
    • 与NaOH溶液反应：Al2O3 + 2NaOH = 2NaAlO2 + H2O
  2. 氢氧化铝是两性氢氧化物：
    • 与盐酸反应：Al(OH)3 + 3HCl = AlCl3 + 3H2O
    • 与NaOH溶液反应：Al(OH)3 + NaOH = NaAlO2 + 2H2O
  3. 实验室制Al(OH)3：Al2(SO4)3 + 6NH3·H2O = 2Al(OH)3↓ + 3(NH4)2SO4
  4. 氢氧化铝加热分解：2Al(OH)3 ≜ Al2O3 + 3H2O （加热）
  5. 往AlCl3溶液中逐滴加入NaOH溶液至刚好生成沉淀：3NaOH + AlCl3 = Al(OH)3↓ + 3NaCl
    继续加入NaOH溶液至沉淀溶解：NaOH + Al(OH)3 = NaAlO2 + 2H2O
  6. 往NaAlO2溶液中加入稀盐酸至刚好生成沉淀：HCl + NaAlO2 + H2O = Al(OH)3↓ + NaCl
    继续加入稀盐酸至沉淀溶解：3HCl + Al(OH)3 = AlCl3 + 3H2O
  7. 向偏铝酸盐溶液中通入过量CO2：CO2 + AlO2- + 2H2O = Al(OH)3↓ + HCO3-
知识点：氧化铝和氢氧化铝都是两性化合物，既能与酸反应也能与碱反应；实验室常用铝盐与氨水反应制备氢氧化铝。

四、铁的重要化合物

附加：氢氧化铁胶体的制备：FeCl3 + H2O（沸水）→ Fe(OH)3（胶体）+ HCl
  1. 氯化铁与氢氧化钠溶液反应：FeCl3 + 3NaOH = Fe(OH)3↓ + 3NaCl
  2. 硫酸亚铁与氢氧化钠溶液反应：2NaOH + FeSO4 = Fe(OH)2↓ + Na2SO4
  3. 氢氧化亚铁在空气中变质：4Fe(OH)2 + O2 + 2H2O = 4Fe(OH)3
  4. 氢氧化铁受热分解：2Fe(OH)3 ≜ Fe2O3 + 3H2O （加热）
  5. 氯化铁溶液中加入铁粉：Fe + 2FeCl3 = 3FeCl2
  6. 氯化亚铁溶液中加入氯水：Cl2 + 2FeCl2 = 2FeCl3
  7. 氧化铁与盐酸反应：Fe2O3 + 6HCl = 2FeCl3 + 3H2O
  8. 氧化亚铁与盐酸反应：FeO + 2HCl = FeCl2 + H2O
  9. 氢氧化铁与盐酸反应：Fe(OH)3 + 3HCl = FeCl3 + 3H2O
  10. 氢氧化亚铁与盐酸反应：Fe(OH)2 + 2HCl = FeCl2 + 2H2O
  11. 氯化铁溶液中加入铜粉：Cu + 2FeCl3 = 2FeCl2 + CuCl2
知识点：Fe²⁺和Fe³⁺可以相互转化；Fe³⁺可用KSCN溶液检验（血红色）；氢氧化亚铁在空气中易被氧化为氢氧化铁。

第四章 非金属及其化合物

一、硅及其化合物

  1. 二氧化硅与氢氟酸反应（雕刻玻璃）：SiO2 + 4HF = SiF4↑ + 2H2O
  2. 硅单质与氢氟酸反应：Si + 4HF = SiF4 + 2H2↑
  3. 二氧化硅与氧化钙高温反应：SiO2 + CaO ≜ CaSiO3 （高温）
  4. 二氧化硅与氢氧化钠溶液反应：SiO2 + 2NaOH = Na2SiO3 + H2O
  5. 单质硅与氢氧化钠溶液反应：Si + 2NaOH + H2O = Na2SiO3 + 2H2↑
  6. 硅酸的制备：Na2SiO3 + 2HCl = H2SiO3（胶体）+ 2NaCl
  7. 硅酸钠溶液通入CO2制得胶体：Na2SiO3 + CO2 + H2O = H2SiO3（胶体）+ Na2CO3
  8. 高纯硅的制备：
    • 二氧化硅与碳高温下反应制粗硅：SiO2 + 2C ≜ Si + 2CO↑ （高温）
    • 粗硅的提纯：Si + 2Cl2 = SiCl4；SiCl4 + 2H2 = Si + 4HCl
知识点：二氧化硅是酸性氧化物，但不溶于水；硅酸是弱酸，可用硅酸盐与强酸反应制备；氢氟酸是唯一能与二氧化硅反应的酸。

二、氯及其化合物

  1. 金属铁在氯气中燃烧：3Cl2 + 2Fe ≜ 2FeCl3 （点燃）
  2. 金属钠在氯气中燃烧：2Na + Cl2 ≜ 2NaCl （点燃）
  3. 氯气与氢气反应：Cl2 + H2 ≜ 2HCl （点燃）
  4. 养金鱼晒水两条方程式：
    • H2O + Cl2 = HCl + HClO
    • 2HClO = 2HCl + O2↑
  5. 氯气的实验室制法：MnO2 + 4HCl(浓) ≜ MnCl2 + 2H2O + Cl2↑ （加热）
  6. 氯气通入澄清石灰水：2Cl2 + 2Ca(OH)2 = Ca(ClO)2 + CaCl2 + 2H2O
  7. 漂白粉原理(失效原因)：Ca(ClO)2 + CO2 + H2O = CaCO3↓ + HClO
知识点：氯气是黄绿色有毒气体，具有强氧化性；氯水成分复杂，具有多种化学性质；漂白粉的有效成分是次氯酸钙。

三、硫及其化合物

  1. 二氧化硫与水反应：SO2 + H2O ⇌ H2SO3
  2. 二氧化硫与氢氧化钙溶液反应：SO2 + Ca(OH)2 = CaSO3↓ + H2O
  3. 工业制硫酸：
    • 二氧化硫与氧气反应：2SO2 + O2 ⇌ 2SO3
    • 三氧化硫与水反应：SO3 + H2O = H2SO4
  4. 等体积的SO2和Cl2通入水中：Cl2 + SO2 + 2H2O = H2SO4 + 2HCl
  5. SO2通入氢硫酸溶液：2H2S + SO2 = 2H2O + 3S↓
  6. 实验室制备SO2：Na2SO3 + H2SO4 = Na2SO4 + H2O + SO2↑
  7. 浓硫酸与铜反应：Cu + 2H2SO4(浓) ≜ CuSO4 + 2H2O + SO2↑ （加热）
  8. 浓硫酸与木炭反应：C + 2H2SO4(浓) ≜ CO2↑ + 2SO2↑ + 2H2O （加热）
知识点：二氧化硫是酸性氧化物，具有漂白性（可逆）；浓硫酸具有强氧化性、脱水性和吸水性。

四、氮及其化合物

  1. "雷雨发庄稼"涉及到的三条方程式：
    • N2 + O2 ⇌ 2NO
    • 2NO + O2 = 2NO2
    • 3NO2 + H2O = 2HNO3 + NO
  2. NO2和氧气4:1比例通入水中完全变成硝酸：4NO2 + O2 + 2H2O = 4HNO3
  3. NO和氧气4:3比例通入水中完全变成硝酸：4NO + 3O2 + 2H2O = 4HNO3
  4. 氨气与水反应：NH3 + H2O ⇌ NH3·H2O ⇌ NH4+ + OH-
  5. 氨水受热分解：NH3·H2O ≜ NH3↑ + H2O （加热）
  6. 氨气催化氧化工业制硝酸：4NH3 + 5O2 = 4NO + 6H2O （催化剂）
  7. 工业合成氨：N2 + 3H2 ⇌ 2NH3 （催化剂）
  8. 氨气实验室制法：2NH4Cl + Ca(OH)2 = CaCl2 + 2NH3↑ + 2H2O
  9. 氨气与氯化氢反应产生白烟：NH3 + HCl = NH4Cl
  10. 碳酸氢铵受热分解：NH4HCO3 ≜ NH3↑ + H2O + CO2↑ （加热）
  11. 硝酸铵与氢氧化钠反应：NH4NO3 + NaOH ≜ NH3↑ + NaNO3 + H2O （加热）
  12. 氯化铵受热分解：NH4Cl ⇌ NH3↑ + HCl
  13. 硫酸铵与氢氧化钠反应：(NH4)2SO4 + 2NaOH ≜ 2NH3↑ + Na2SO4 + 2H2O （加热）
  14. 浓硝酸与铜反应：Cu + 4HNO3(浓) = Cu(NO3)2 + 2H2O + 2NO2↑
  15. 稀硝酸与铜反应：3Cu + 8HNO3(稀) ≜ 3Cu(NO3)2 + 4H2O + 2NO↑ （加热）
知识点：氮的氧化物有多种价态，可相互转化；氨气易溶于水，水溶液呈弱碱性；硝酸具有强氧化性，浓度不同还原产物不同。
学习建议：化学方程式的学习要结合实验现象和反应条件；注意区分相似反应的不同条件；掌握离子方程式的书写方法；构建知识网络，理解物质间的转化关系。
